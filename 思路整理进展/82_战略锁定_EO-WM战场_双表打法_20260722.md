# 82 · 战略锁定：EO-WM 战场 + 双表打法（机制尺子翻盘）2026-07-22

> 依据：亲自抽取 EO-WM(arXiv 2606.27277, 2026-06 预印本) 正文 + Table 2 数字 + 服务器现状核实。**这是"保留世界模型 + 够得着顶端"的现实主线。**

## 0. 一句话
**别在 Contextformer 的 R²/RMSE 尺子上硬碰（我们 B/C、还输 climatology）；去 EO-WM 已经建好的"机制尺子"上打——那把尺子上，R² 榜的强者(PredRNN/SimVP)因过度平滑而崩，我们大概率更能打。**

## 1. EO-WM 是什么（同类先例，坐实）
- **arXiv 2026-06 预印本**（未确认已发表；参考文献里有 ICCV'25 Workshop 同类 EO 预测工作 = 该方向有落点）。
- **video diffusion transformer（MMDiT）+ 物理条件化**的 EO 世界模型。
- **数据/协议 = EarthNet2021，10→20**（**和我们完全相同**）。
- **核心论点（原话）**："标准 EarthNet2021 指标只问是否匹配一个已实现的未来，**捕捉不到世界模型能力**" → **自造两个诊断 benchmark**。
- **对手 = Earthformer + OpenSTL(SimVP/TAU/PredRNN/PredRNNv2/PhyDNet) + Latte/Wan2.1**——**不含 Contextformer**。
- **它怎么赢**：ENS/NDVI-MAE 只求"有竞争力"，靠 **TN-MAE 全 severity 最优 + DHR/PDC 最优** 立住；**自己承认**"增采样提升重构指标却降 PDC"——即**主动接受"重构 vs 机制"的权衡，不 claim 像素 SOTA**。

## 2. 决定性证据：机制尺子上排名翻盘（Table 2 Extreme Summer，我抽到的真数）
| 模型 | ENS↑ | NDVI-MAE↓ | 备注 |
|---|---|---|---|
| Earthformer | **0.2539** | **0.1099** | 确定性里最好 |
| 生成式(某) | 0.2028 | 0.1345 | |
| PredRNN v2 | 0.1435 | 0.2358 | |
| SimVP | 0.1177 | 0.1846 | |
| PredRNN | 0.0589 | **0.3757** | **R²榜0.62的强者，在极端NDVI-MAE上崩** |

**关键洞察**：**R²/RMSE 榜的赢家(PredRNN 0.62 / SimVP 0.60)是靠"过度平滑=预测均值"赢的——一到极端事件/NDVI-MAE 就崩**（PredRNN NDVI-MAE 0.38、SimVP 0.18）。**Earthformer(R²最弱0.52)反而在机制尺子上最好(0.11)。→ 机制尺子和 R² 尺子的排名是反的。**

**我们的位置（利好信号，但未 apples-to-apples）**：S1a 的 NDVI-MAE≈**0.10**（val_dev），**落在 Earthformer 档、优于 SimVP/PredRNN**。因为我们的残差头锚定 last-valid、学变化量，**不是过度平滑派** → 在机制尺子上大概率比在 R² 尺子上好看得多。
> ⚠️ **诚实边界**：我们的 0.10 是 GreenEarthNet val_dev，EO-WM 的 0.11 是 EarthNet2021 extreme_test，**不同 split、暂不可直接比**。**必须把我们的模型跑到 extreme_test/seasonal 上、用它的指标，才算数。**

## 3. 双表打法（锁定）
- **Table A（标准精度，诚实次要）**：GreenEarthNet ood-t R²/RMSE = 0.5472/0.1795（"有竞争力、非SOTA"，一行带过）。
- **Table B（世界模型机制，主战场）**：**采用 EO-WM 的两个 benchmark**：
  - **Extreme Summer**：TN-MAE(谷值NDVI-MAE) + DAE(下降幅度误差)，分 low/mid/high severity。← 对应我们**冻结状态植被衰退事件读出**。
  - **Seasonal Matched-Pair**：DRR/DHR(方向命中)/PDC。← 对应我们 **composed 一致性 + 观测因子化(matched-pair)**。
  - 对手：Earthformer/生成式（够得着），**外加我们自己的 Direct(S1a) 作 ablation**。
