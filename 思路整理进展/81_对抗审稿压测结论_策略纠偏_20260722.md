## 1. 单一最大风险（综合原话）
**还没跑那两个决定"有没有 paper"的廉价度量**：(1) S1a 的 ood-t R²/RMSE；(2) S1a 上 post-hoc composed(5+5) vs direct(10) 的 gap。最坏复合情形真实存在：精度大概率 tier B/C（≈0.52–0.60，不比自己的 Direct-P4 copy 强），且官方 ENS 输给静态 persistence → 全靠接口证据扛；**若 post-hoc gap 大**，唯一退路是 S1b 一致性微调（训练 composed=direct 再当"涌现"报 = 循环论证），恰在弱精度让你无其他腿可站时。

## 2. 我上一轮哪里对、哪里必须改
- **对**：Table3(i) post-hoc **确实能在 S1a Direct 上做**——`ObsWorldDirectPathModel` 拥有共享 `ControlledTransition`+`core`，partition_start=0 处 ~50 行脚本即可，**无 rollout、无重训**（审稿人说"架构不可能"是错的，综合已澄清）。
- **必须改**：光报 gap 没用甚至危险。**gap 小可能是平凡塌陷**（residual_scale 太小 → next≈state → 两路都贴 state0 → gap 好看但状态没动）。必须同时报 4 护栏：
  1. `state_delta_norm / state_std`：证明 T 真的推动了状态（防近恒等/低秩塌陷）。
  2. `pred_direct` 与 `pred_composed` **各自**对真值 endpoint 精度（DirectPath 里 10 天 endpoint 有监督）→"两支都准且一致"，非"两个平庸预测互相同意"。
  3. shuffle 配对下限 + 用样本间潜距归一化 gap。
  4. **多深度曲线**：T15 vs 5+5+5、非均分 3+7/4+6。只报单点 10=5+5 是最弱的半群证据，正招"为何只 2 步/停在还能 work 的地方"质疑。

## 3. 叙事重心迁移（最可辩护框架，综合原话提炼）
**不把 ObsWorld 定位成 SOTA 预报器，而是"一个强 Direct 地表预报器的可证伪接口研究"。**
- **头条 = 可组合转移的后验测量**：`T(10d) ≈ T(5d)∘T(5d)`，**无一致性训练**，对 shuffle 下限验证、按潜距归一、证明状态被推动（非近恒等）、两 endpoint 各自准；**组合边界公开报告**（≥3 步退化 = 正好是"短程转移"的既定范围，不是藏失败）。
- **精度 = 有竞争力、相对 plain Direct 骨架零接口回归**（3 seed 拿不到前**绝不写 SOTA**）。
- **φ 诚实展示为"保留采集信息、非采集不变"**（正好印证 76 的 no-invariance disclaimer）。
- **L1C/L2A 明确列 future work**，不靠软化摘要偷渡强 O-operator 读法。
- 精度与 ENS-分量诊断作**支撑**，不作头条。

## 5. 两个立即的诚实/事实修正

## 6. 其他必做（综合 ranked_actions 摘录）
- **physical4 是 oracle 外生输入**，direct/composed **同样喂入** → 部分一致性由共享驱动天然保证；加 shuffled-weather 对照，披露它 bound 了"自主动力学 T"的说法。
- **ENS 分量分解**（MAD/OLS/EMD/SSIM，走 `--official-score`）定位损失在 SSIM^10.3 项，展示在植被相关分量上取胜；引 Benson-2024 自己的 ENS/SSIM 批评，而非空口说"伪影"。
- **"sufficient statistic" 要挣或删**：加 screening-off 探针（frozen z vs frozen[z+raw-history]），等价 = 充分性证据；不跑就改名"reusable, composition-consistent latent forecast state"，全文删 "sufficient statistic"。Table3(ii) 基线从 raw-history 换成 λ=0 / 等容量 plain ViT-S。
- **controllability 探针（近免费）**：`ControlledTransition` 自带 `use_D/use_G/use_h` 干预开关，翻 use_D 或清零 φ 看状态/读出是否合理移动；挣"controllable"腿，否则从四属性头条里删"stable/controllable"。
- **全文 grep 对齐**：`L1C|L2A|SAR|physical state|digital twin|first weather world model|sufficient`；re-score 0.5243 anchor 于同一 ood-t harness；明说"仅 ood-t split"。
